스타벅스 마케팅 데이터 분석 파이프라인
│
├── 📦 adff1.ipynb — 전처리
│   ├── 입력: portfolio.csv / profile.csv / transcript.csv / starbucks_menu.csv
│   ├── Portfolio   채널 one-hot encoding, reward_ratio, offer_strength 파생
│   ├── Transcript  value 컬럼 파싱, day 파생
│   │               viewed_before_complete 플래그 생성
│   ├── Profile     age 이상치(118→NaN), 결측 플래그, 날짜 변환
│   ├── 병합        transcript ← portfolio ← profile
│   └── 출력: Final.csv (306,534행 × 24컬럼)
│
├── 🔍 eda.ipynb — 기본 EDA + 파생변수
│   ├── 입력: Final.csv
│   ├── 파생변수    join_year/month/cohort, age_group, income_group, age_gender
│   ├── 기본 현황   이벤트 분포, 퍼널 수치(수신→열람→완료), 미인지 완료율
│   └── 고객 프로파일
│       ├── 성별      M 49.9% / F 36.1% / Unknown 12.8%
│       ├── 연령대    평균 54.4세, 60대 이상 34.6%로 최다
│       ├── 소득      5-7.5만 37.9%로 최다, 평균 $65,405
│       └── 거래금액  평균 $12.78 / 중앙값 $8.89 / 최대 $1,062
│       출력: final_eda.csv (306,534행 × 31컬럼)
│
├── ⏱️ stage1.ipynb — 시간 흐름 분석 (Act 1)
│   ├── 입력: final_eda.csv
│   ├── funnel_instance 생성
│   │   └── merge_asof로 received→viewed→completed 인스턴스 단위 매핑
│   ├── 분석 1  수신→첫열람  평균 1.1일 / 24시간 내 66.0%
│   │          수신→첫완료  평균 2.5일 / 7일 내 96.7%
│   ├── 분석 2  duration 내 완료율 100% (유효기간 설계 적절)
│   ├── 분석 3  자연 거래 vs 오퍼 중 거래
│   │          자연 거래 12.3% / 금액 차이 $0.02 (거의 없음)
│   └── 출력: funnel_instance.csv (76,277행)
│
├── 📊 stage2.ipynb — 퍼널 통계 분석 (Act 2)
│   ├── 입력: funnel_instance.csv + final_eda.csv
│   ├── 퍼널 전환율  수신→열람 75.7% / 열람→완료 57.3%
│   ├── 검정 1  bogo vs discount 반응 시간
│   │          Mann-Whitney: bogo가 0.8일 빠름 (p<0.001, 중간 효과)
│   ├── 분석 2-1  미열람 고객 24.3%
│   │            저소득(5만미만) 과대 대표, discount/informational 미열람률 높음
│   ├── 분석 2-2  미인지 완료 17.0%
│   │            거래금액 오히려 더 높음 ($20.37 vs $18.05, p<0.05)
│   │            → 오퍼 의식 없이도 자연 충족하는 충성 고객
│   └── 분석 2-3  informational 거래 연결률 54.8%
│                bogo/discount 완료율과 유사 수준
│
└── 🧩 stage3.ipynb — 세그먼트 분석 (Act 3)
    ├── 입력: funnel_instance.csv + final_eda.csv
    ├── 분석 3-1  소득 구간별 (카이제곱 V=0.379, 강한 효과)
    │            완료율: 5만미만 43.8% → 10만이상 76.5%
    │            거래금액: $5.79 → $26.09 (모든 쌍 Tukey 유의)
    ├── 분석 3-2  성별 × 오퍼 유형
    │            여성이 남성보다 bogo +19.3%p / discount +14.0%p 높음
    ├── 분석 3-3  연령대 × 오퍼 유형 (효과 크기: 중간)
    │            나이 많을수록 완료율 ↑
    │            discount가 bogo보다 모든 연령대에서 완료율 높음
    └── 분석 3-4  가입년도별
                2016년 완료율 최고 73.0% / 2018년 최저 34.2%
                신규 가입자일수록 오퍼 반응 낮음
