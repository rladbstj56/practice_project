## 분석 구조

### A안 — CRM 캠페인 효과 분석

```
1막 시간 흐름(언제 반응하는가)        오퍼 발송 후 반응 시간 / 자연 거래 패턴
    ↓
2막 퍼널 분석(어디서 이탈하는가)       미열람 고객 / 미인지 완료 / informational 효과
    ↓
3막 세그먼트(누가 반응하는가)         소득·성별·연령대·채널·황금기별(가장 소비를 활발하게 하는 기간- 통계 분석 해봐야 함) 완료율
    ↓
전략 제안            4개 세그먼트 맞춤 프로모션 전략
```

**질문**
- 오퍼를 받았는데 열람조차 안 한 고객이 얼마나 되는가?
- 어떤 고객군이 황금기(가입 몇년차)에 가장 폭발적으로 반응하는가?
- discount vs bogo, 어떤 오퍼가 더 효과적인가?
- 4채널 발송이 2채널보다 실제로 완료율이 높은가?

### B안 — 고객 가치(LTV) 파레토 분석

```
파레토 분석 : "전체 결과의 대부분은 소수의 원인에서 나온다"는 80/20 법칙을 데이터로 검증하는 분석. 
고객을 총 거래금액 기준으로 내림차순 정렬한 뒤 누적 매출 비율을 계산하면, 상위 20% 고객이 전체 매출의 상당 부분을 차지하는 패턴을 확인할 수 있다. 
이를 통해 마케팅 예산을 어떤 고객군에 집중해야 ROI가 높아지는지 판단할 수 있다

```

고객별 집계          총 거래금액 / 거래 빈도 / 오퍼 완료 횟수
    ↓
파레토 검증          상위 20% 고객이 전체 매출의 몇 %를 차지하는가
    ↓
고객 등급화          High / Mid / Low Value 3등급 분류
    ↓
전략 연결            A안 황금기 인사이트와 결합 → 투자 우선순위 결정

```

**핵심 질문**
- 상위 20% 고객이 전체 매출의 얼마를 차지하는가?
- High Value 고객의 인구통계적 특성은?
- HV 고객은 오퍼 없이도 구매하는가?

| 대시보드 | 스토리 | 핵심 차트 |
